
Image Encryption & Decryption Tool
==================================

How to run:

1. Open terminal or Anaconda prompt
2. Navigate to this project folder
3. Install required packages:

   pip install streamlit pillow opencv-python pycryptodome numpy

4. Run the Streamlit app:

   streamlit run app.py

5. Use the web interface to:
   - Upload an image
   - Encrypt and generate encrypted.bin and key.bin
   - Decrypt the image from encrypted.bin using key.bin

All files are saved in the 'output/' folder.
